# Stopwatch (3s Display Steps)
A real-time stopwatch where every 1 second of real time equals +3 seconds on display.

**Live Demo:** https://aorbo1815.github.io/stopwatch-3s/

## Features
- 0, 3, 6, , 30 display (increments every second)
- Real-time counter under the bar
- Auto-stops at 30 seconds unless reset
- Start / Stop / Reset buttons

## Tech
HTML, CSS, JavaScript
